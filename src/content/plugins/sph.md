做视频号的朋友，一定遇到过这些头疼时刻：

- 积累了上百条视频，想统一调整可见权限，只能一条一条手动改
- 早期发的内容质量不高，想批量清理，翻页翻到手抽筋
- 图文和视频混着管理，操作路径还不一样，找半天找不到入口

如果你也有这些困扰，今天分享一个我自做自己用的浏览器插件。

---

## 它能做什么？

插件安装后，打开视频号创作者后台，页面右侧会自动出现一个浮动管理面板。核心功能就三个：

**🔓 批量公开**
把选中的内容从「仅自己可见」一键设为「公开可见」。适合先把内容准备好，再统一发布。

**🔒 批量私密**
把选中的内容从「公开」一键设为「仅自己可见」。不想删但暂时不想展示的内容，先藏起来。

**🗑️ 批量删除**
勾选要删的内容，一键删除。默认开启二次确认，防止误删。

## 支持哪些页面？

| 页面 | 支持的操作 |
|------|-----------|
| 视频列表 | 公开 / 私密 / 删除 |
| 图文列表 | 公开 / 私密 / 删除 |
| 草稿箱 | 删除 |

三个页面通用同一个面板，插件会自动识别当前在哪个页面，显示对应的操作按钮。

---

## 几个实用细节

**操作间隔可调**
批量操作之间会自动加延迟（默认 0.8 秒），避免操作太快触发平台风控。间隔时间可以在设置面板里自由调节，范围从 0.5 秒到 3 秒。

**随机抖动防检测**
每次操作的延迟不是固定值，会自动叠加 ±5% 的随机抖动，模拟人工操作节奏。

**自动翻页**
勾选了「自动翻页」后，插件会自动翻到下一页继续处理，不需要手动翻页。可以设置最大翻页数，防止跑到太远。

**随时中断**
操作过程中可以随时点「停止」按钮中断，已处理的内容不会回滚。

**删除前确认**
默认开启二次确认弹窗。如果你信任自己的操作，也可以在设置里关掉，直接执行。

## 怎么用？

1. 下载插件文件（没发商店，直接开发者模式加载，把文件夹拖进去就行）
2. 打开 `chrome://extensions/`，开启「开发者模式」
3. 点「加载已解压的扩展程序」，选择插件目录或者直接
4. 打开视频号创作者后台，进入内容管理页面
5. 页面右侧出现浮动面板，勾选内容后点击操作按钮即可

---

## 为什么做这个插件？

视频号创作者后台本身没有批量管理功能，内容多了之后管理起来非常低效。市面上也没有找到好用的第三方工具，所以自己动手做了一个。

插件基于 DOM 模拟点击实现，不需要登录授权、不需要 API 接口，直接在页面上模拟人工操作。所有数据都在本地处理，不会上传到任何服务器。
